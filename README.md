# Webapp Boilerplate

### Use this boilerplate as template

![Use this Template](assets/image/use-template.png)

### Installation

```bash
# Install node dependencies
$ npm install
```

### Run application

```bash
# Before the first run, you have to compile the app
$ npm run compile

# Run application
$ npm run start
```

```bash
# Run application in developer mode
$ npm run dev
```
